# Islam Atef - Professional Portfolio

This is the source code for my professional portfolio website, showcasing my expertise as a Full Stack Developer and Security Researcher.

## Features
- **Cybersecurity Dark Theme:** A premium UI built with React, Vite, and Vanilla CSS.
- **Responsive Design:** Fully responsive across all devices.
- **Glassmorphism:** Modern aesthetics emphasizing a hacker/security vibe.

## Technologies
- React (Vite)
- CSS (Custom properties, animations)
- Lucide React (Icons)

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server

## Deployment
This project is configured to be automatically deployed via Netlify whenever changes are pushed to the `main` branch.
